# Pong Game 🏓

A fully functional Pong game developed using Python's Turtle Graphics module. This project recreates the classic arcade experience with two-player gameplay, responsive paddle controls, ball physics, collision detection, and a live scoring system.

# Features
- Two-player gameplay
- Smooth paddle movement
- Ball bounce mechanics
- Wall and paddle collision detection
- Real-time score tracking
- Object-Oriented Programming (OOP) structure
- Modular code organization

# Technologies Used
- Python 3
- Turtle Graphics
